This is for plotting the seismograms when using different PML boundary conditions for the Marmousi2 model test.

***RUNNING DESCRIPTION
1. Input files
The input files are named as "**PML.txt." for the time-domain wavefield results.

2. Figure plotting
Run the "plot_seismograms.m" program to get the Figures Figures 19 and 20 in our GEOPHYSICS paper.

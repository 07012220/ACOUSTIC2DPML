This is for plotting both the time-domain and frequency-domain snapshots using different PML boundary conditions for the Marmousi2 model test.

***RUNNING DESCRIPTION
1. Input files
The input file "Marmousi2.mat" is for the time-domain and frequency-domain wavefield results.

2. Figure plotting
Run the file "snapshot.m" to get the Figures 21 and 22 in our GEOPHYSICS paper.
